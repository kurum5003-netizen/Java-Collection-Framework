# Java-Collection-Framework
Bu repo, Java Collection Framework yapısını anlamak, uygulamak ve gerçek senaryolarda nasıl kullanılacağını göstermek amacıyla hazırlanmış örnek projeleri içermektedir. Her proje; List, Set, Map, Queue, Stack gibi temel koleksiyon türlerinin kullanımını, performans farklarını ve uygun veri yapısını seçme pratiklerini göstermeyi hedefler.

- **ArrayList Kullanımı**

- **LinkedList ile İşlem Sırası Yönetimi**

- **HashSet / TreeSet ile Benzersiz Veri Yönetimi**

- **HashMap / LinkedHashMap / TreeMap ile Anahtar-Değer Yapıları**

- **Queue & Deque Yapıları**

- **Stack ile LIFO Mantığı**

- **Comparator ve Comparable ile Sıralama**

- **Collections Utility Sınıfı ile Yardımcı Fonksiyonlar**


---

## Örnek Projeler
- **Harf frekansı bulma**
- **Macera oyunu**
- **Stack yardımı ile palindrom kontrolü**
- **Queue kullanarak sıra kontrolü**
- **Itabre interface ile kumanda yapımı**
- **Priority Queue ve Comparable interface ile hastane uygulaması**
